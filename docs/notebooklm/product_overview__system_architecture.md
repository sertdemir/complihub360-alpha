Experience Layer (UI)\n\n\n4.1 Screens\n\n\nLanding (Country selection + Search + Optional category selection)\n\n\nWizard (Context sharpening, 4–5 steps)\n\n\nResults (Tabbed results view; providers prominent on right sidebar)\n\n\nPrimary CTA Modal (engagement request)\n\n\nConfirmation / Status view\n\n\nDashboard (registered users only)\n\n\n4.2 Interaction Model\n\n\nResults are displayed tab-based.\n\n\nThe wizard is optional; the provider grid remains visible to support fast users.\n\n\nEditing and advanced functions are available only in the dashboard.\n\n\n\n\n\n5.
